# Sephirót Kalfírvach

Diez nombres en Kalfírvach v0.4 para los diez arquetipos
deléficos del Árbol de la Vida. Cada nombre es una creación
original que captura la esencia unificadora de todas las
deidades listadas, siguiendo las reglas de composición nominal,
derivación y fonotáctica de la lengua.

---

## 1. KETHER — La Corona
**Ékashúnya** /e.kaˈʃuɲ.ja/

| | |
|---|---|
| **Composición** | éka (uno) + shúnya (vacío/potencial) |
| **Estructura** | Modificador + Núcleo |
| **Glosa literal** | "Vacío-Uno" / "El Vacío Uno" |
| **Arquetipo** | La Fuente indiferenciada, el Absoluto antes de toda división |
| **Deidades** | Zeus, Ymir, Wuji, Atum, Urano, Brahman, Paramashiva, Vishnu (preservador), Dios Padre, El Uno hermético |

El Vacío Uno. No es vacío como ausencia — es el potencial puro,
el *śūnya* fecundo del que todo emana. La unidad que contiene
todas las contradicciones sin resolverlas. Ni creador ni creación:
la posibilidad misma de ambos.

---

## 2. CHOKMAH — La Sabiduría
**Nádahíkma** /na.daˈhik.ma/

| | |
|---|---|
| **Composición** | náda (sonido primordial) + híkma (sabiduría) |
| **Estructura** | Modificador + Núcleo |
| **Glosa literal** | "Sonido-Sabiduría" / "La Sabiduría que Suena" |
| **Arquetipo** | El Logos, la sabiduría activa que crea mediante la Palabra |
| **Deidades** | Dakshinamurti, Brahma, Thoth, Ptah, Poseidón, Prometeo, Enki, Odín, Yang, Ahura Mazda |

El Logos en su sentido más profundo: el principio activo que
organiza el caos en cosmos mediante vibración, sonido y palabra.
Es la inteligencia que todo lo penetra — el Fiat Lux anterior
a la luz misma. *Náda* es el sonido primordial del que nacen
todos los mantras; *híkma* es la sabiduría que ordena.

---

## 3. BINAH — El Entendimiento
**Sírrshákta** /sirˈʃak.ta/

| | |
|---|---|
| **Composición** | sírr (secreto/misterio) + shaktí (poder) |
| **Estructura** | Modificador + Núcleo |
| **Glosa literal** | "Poder Secreto" / "El Poder del Misterio" |
| **Arquetipo** | La Madre Receptiva, la comprensión profunda, el útero de todas las formas |
| **Deidades** | Jehová Elohim, Aima Elohim, Espíritu Santo, María, Shakti, Parvati, Saraswati, Ma'at, Isis, Hathor, Deméter, Hera, Atena, Tiamat, Frigg, Yin, Sofía |

El poder que no se impone — *recibe*. El misterio fecundo que
concibe en su seno la semilla de Chokmah y la gesta en formas
concretas de comprensión. *Sírr* no es secreto que oculta, sino
misterio que invita a adentrarse. *Shákta* es la potencia femenina
que todo lo contiene y lo transforma.

---

## 4. CHESED — La Misericordia
**Súpréma** /suˈpre.ma/

| | |
|---|---|
| **Composición** | su- (aumentativo "gran") + prem (amor) + -ma (resultado manifestado) |
| **Estructura** | Prefijo + Raíz + Sufijo derivativo |
| **Glosa literal** | "Gran Amor Manifestado" |
| **Arquetipo** | El amor expansivo, la gracia, la misericordia que abraza sin límite |
| **Deidades** | Gedulah, El, Cristo, San Francisco, Vishnu/Krishna, Laksmi, Horus, Bastet, Freyja, Brigid |

El Amor no como emoción sino como **principio expansivo del ser**:
la fuerza que acerca, conecta, y sostiene la creación en unidad.
*Súpréma* es la manifestación concreta del amor divino — el acto
de gracia que no juzga sino que envuelve.

---

## 5. GEBURAH — El Rigor
**Áshakrod** /a.ʃaˈkrod/

| | |
|---|---|
| **Composición** | ásha (verdad/orden cósmico) + krod (ira santa) |
| **Estructura** | Modificador + Núcleo |
| **Glosa literal** | "Ira de la Verdad" / "Furia Cósmica" |
| **Arquetipo** | El juicio divino, la severidad que destruye la ilusión, el límite sagrado |
| **Deidades** | Pachad, Din, Arcángel Miguel, Shiva destructor, Kali, Durga, Sekhmet, Ares, Nergal, Thor, Némesis, Angra Mainyu, Huitzilopochtli |

No es ira ciega — es **la furia del orden cósmico contra lo que
se opone a la verdad**. Geburah es el límite que toda forma necesita
para ser lo que es. *Ásha* es la verdad-estructura del cosmos
(hermana del *Rta* védico y el *Ma'at* egipcio). *Krod* es la
ira que purifica, la espada que separa lo real de lo ilusorio.

---

## 6. TIPHARETH — La Belleza
**Nákhzírsól** /naxˈzir.sol/

| | |
|---|---|
| **Composición** | nakh-zir (brillante) + sól (sol/luz solar) |
| **Estructura** | Modificador + Núcleo |
| **Glosa literal** | "Sol Radiante" / "El Brillante Solar" |
| **Arquetipo** | El hijo, el centro, la belleza armónica, la luz que se sacrifica |
| **Deidades** | Melekh, Tzaddik, Cristo, Krishna, Ra, Apolo, Horus, Helios, Shamash, Baldr, el Sol alquímico |

El arquetipo central del Árbol. *Nákhzír* es el resplandor
consciente de quien sabe que es luz; *sól* no es el astro físico
sino la luz solar como principio — la revelación que ilumina
y se revela. Tiphareth es el corazón que sostiene el equilibrio
entre Chesed y Geburah, entre la misericordia y el juicio.

---

## 7. NETZACH — La Victoria
**Ékshákti** /ekˈʃak.ti/

| | |
|---|---|
| **Composición** | ék (éxtasis/trascendencia) + shaktí (poder) |
| **Estructura** | Modificador + Núcleo |
| **Glosa literal** | "Poder Extático" / "Fuerza de Trascendencia" |
| **Arquetipo** | La victoria eterna, la fuerza instintiva, la potencia natural y extática |
| **Deidades** | Oz, Espíritu Santo, San Jorge, Indra, Ganesha, Sekhmet, Neith, Artemisa, Dioniso, Istar, Freyr, Cernunnos, Xochiquétzal |

*Ék* — del griego *ékstasis* "estar fuera de sí" y el egipcio
*akh* "espíritu transfigurado" — es la potencia que rompe los
límites de la forma establecida. Netzach es la fuerza que vence
porque no reconoce obstáculo: la vitalidad pura de la naturaleza,
el éxtasis dionisíaco, la flecha de Artemisa que siempre da en
el blanco.

---

## 8. HOD — La Gloria
**Mántravácha** /man.traˈva.tʃa/

| | |
|---|---|
| **Composición** | mántra (palabra-poder) + vacha (habla/decir) |
| **Estructura** | Modificador + Núcleo |
| **Glosa literal** | "Palabra de Poder Declarada" / "El Decir del Mantra" |
| **Arquetipo** | El intelecto comunicante, la gloria del verbo, la traducción de lo divino a lo inteligible |
| **Deidades** | Metatron, Gabriel, San Juan Evangelista, Ganesha como sabiduría, Odín, Nabu, Hermes Trismegisto, Mercurio, Samael |

El esplendor del intelecto que **articula** lo recibido. Hod es
el poder de dar forma verbal al misterio. *Mántra* es la palabra
que transforma; *vácha* es el acto mismo de hablarla. Donde
Chokmah es el Verbo que crea, Hod es el Verbo que **comunica**,
que traduce la sabiduría en símbolos que la mente puede procesar.

---

## 9. YESOD — El Fundamento
**Samúndríga** /saˈmun.dri.ɣa/

| | |
|---|---|
| **Composición** | samunda (océano de conciencia) + ríga (conciencia individual) |
| **Estructura** | Modificador + Núcleo |
| **Glosa literal** | "Conciencia Oceánica" / "Océano de Consciencia" |
| **Arquetipo** | La fundación, el mundo lunar y onírico, el receptáculo de todas las imágenes |
| **Deidades** | Virgen María, Chandra, Thoth, Selene, Sin, Máni, Cerridwen, la Luna, Shaddai El Chai, Hécate |

Yesod es el plano astral, el tesoro de imágenes donde todo lo que
existe en los planos superiores cobra forma sutil antes de
manifestarse. *Samúnda* es el océano infinito de conciencia que
todo lo conecta; *ríga* es la ola individual, la conciencia que
emerge de ese océano. Entre ambos se teje la trama de los sueños,
la memoria, la imaginación y el deseo.

---

## 10. MALKUTH — El Reino
**Rúpagétik** /ru.paˈɣe.tiɡ/

| | |
|---|---|
| **Composición** | rupa (forma/plano material) + gétik (material/físico) |
| **Estructura** | Modificador + Núcleo |
| **Glosa literal** | "Forma Material" / "Cuerpo de la Creación" |
| **Arquetipo** | El mundo físico, el reino de la manifestación, la Shekinah inmanente |
| **Deidades** | Adonai Ha Aretz, Prithivi, Vishnu/Varaha, Geb, Taweret, Gaia, Deméter, Ki, Jörð, Tláloc |

El Reino no es una caída — es la culminación del proceso
emanativo. *Rúpa* (del sánscrito) es la forma que hace visible
lo invisible; *gétik* (del persa *gētīg*) es el mundo material
como manifestación sagrada. Juntos afirman que **lo físico es el
cuerpo de lo divino**, el templo donde la creación completa su
ciclo y la luz se hace tierra fértil.

## Principios de construcción

Todos los nombres siguen las reglas de Kalfírvach v0.4:

- **Composición**: Modificador + Núcleo (salvo donde se usan prefijos/sufijos derivativos)
- **Acento**: siempre en la penúltima sílaba del compuesto entero
- **Fonotáctica**: sin fonemas ajenos al inventario (21 consonantes, 5 vocales)
- **Derivación**: afijos del §3.6 (su- aumentativo, -ma resultado, etc.)
- **Raíces**: documentadas en el léxico v0.4 o en la gramática v0.4 (§3.3.2 para -sól, §2.2.3 para -ék)
- **Justificación**: cada raíz tiene origen en ≥2 lenguas fuente (sánscrito, persa, avéstico, griego, egipcio, árabe, tibetano)

No se trata de "traducir" Zeus ni Shakti ni Cristo. Se trata de
**nombrar el arquetipo que todos ellos comparten** en la lengua
que mejor puede expresar esa unidad: Kalfírvach.
