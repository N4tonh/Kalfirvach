# Textos Modelo de Kalfírvach — v1.3

> **Versión:** v1.3
> **Fecha:** 2026-06-06
> **Estado:** canónico. Estos textos son la **prueba de uso real** del sistema v1.3 (F2 fonosemántica, F3 actitudinales coercitivos, F4 vocativo, F5 léxico esotérico). Sirven para verificar la coherencia cross-doc entre gramática, fonología y lexicón al cierre del ciclo de audit v1.3.

Estos dos textos modelo invocan los frentes centrales de v1.3. La **invocación 1** combina F3 + F4 + F2 en una sola cláusula canónica; la **invocación 2** (en `textos_modelo_v1.3.md`, parte 2) agrega F5 para demostrar el uso completo del instrumental ritual con la pragmática coercitiva.

> **Convención de glosa:** LA con guiones marca fronteras morfológicas; las mayúsculas etiquetan categorías (PRES, PAS, HIER, INF, DEF.SG, 1sg, etc.); las minúsculas son palabras léxicas. La grafía KFA es la politónica con letras arcaicas (san ϻ, sampi ϡ, digamma ϝ, yot ϳ, heta ͱ).

## Invocación 1 — F3 + F4 + F2 (canónica)

**Cláusula ritual canónica (v1.3, la que ejercita F2, F3 y F4 en una sola pieza):**

```
o Yaho, bīj raka wah-nam-ran-poi
```

**Gloss didáctica (palabra por palabra):**

| Morfema | Categoría | Gloss |
|---------|-----------|-------|
| `o` | partícula (§1.7 vocativo, F4) | vocativo, invocación dirigida |
| `Yaho` | nombre propio (lexicón `onomastica`) | YHWH, el nombre revelado |
| `,` | ortográfico (no gramatical) | pausa ceremonial |
| `bīj` | sustantivo (lexicón `magia_y_mitologia`) | semilla, bīja-marcador |
| `raka` | bīja nativo (fonología §11.2 worked example A) | "el fuego sellado en el tiempo" (`/r/` fuego + `/a/` astral + `/k/` Saturno) |
| `wah` | evidencial (§3.13, "revelado") | modo de conocimiento: visionario, jerárquico |
| `-nam` | actitudinal (§2.2, "reverencia") | respeto ante lo sagrado |
| `-ran` | actitudinal coercitivo (§5.3, F3 "precativo") | ruego, súplica respetuosa |
| `-poi` | performativo (§5.2) | modo declarativo performativo: "performativamente" |

**Gloss corrida:** "¡Oh YHWH, performativamente, con reverencia te ruego, invoco la semilla fuego-cronos con visión revelada!"

**Notas gramaticales:**

- **§1.7 vocativo:** la partícula `o` ocupa posición pre-nominal absoluta (`o + SN + , + resto de cláusula`). El SN `Yaho` (nombre propio revelado) no lleva evidencial ni postposición de caso; la coma es ortográfica, no gramatical. La curva prosódica vocativa (rise-fall en el nombre invocado + caída ceremonial, fonología §9.3) se dispara morfológicamente por la presencia de `o`.
- **§5.3 actitudinal `-ran`:** el sufijo `-ran` codifica fuerza ilocutiva **precativa** (el más bajo de los cinco actitudinales coercitivos: `-ran` < `-ro` < `-ro-hék` < `-rak` < `-rab`). En este ejemplo se combina con el evidencial hierático `-wah` (revelado) y el evidencial reverencial `-nam` (reverencia), formando la cadena ritual estándar de invocación respetuosa.
- **Fonología §11.2 worked example A (bīja `raka`):** `raka` se genera por el algoritmo fonosemántico: raíz `/r/` (fuego/sol, §11.1 matriz) + vocal `/a/` (astral layer) + sufijo planetario `/k/` (Saturno/cronos, §11.1). La forma simplifica a bisílabo CV-CV por silabificación KFA (§10). Gloss semántica: "el fuego sellado en el tiempo" — el `-k/` saturnal es lo que hace que el bīja tenga **persistencia temporal**, requisito de toda invocación operativa (no se invoca "ahora y se olvida"; se invoca "ahora y queda sellado").
- **Cross-ref:** fonología §9.3 (curva vocativa disparada por `o`), §11.0-§11.4 (sistema fonosemántico), gramática §1.7 (vocativo), §2.2 (actitudinales), §3.13 (evidenciales), §5.3 (jerarquía coercitiva), lexicón `o` (vocativo), `bīj` (bīja-marker), `Yaho` (nombre revelado), actitudinales `-ran` y `-nam` (F3).

## Invocación 2 — F3 + F4 + F5 (operativa plena)

**Cláusula ritual operativa (v1.3, F3 + F4 + F5 con instrumental completo):**

```
o Yaho, altara-te qadesh kár-a-h-wah-nam-rab-poi, lampa-ka virga-ka athama-ka tunika-ka, halel.
<!-- CORRECCIÓN v1.3: 5 correcciones gramaticales — dativo altar, evidencial wah, hierático h-nam, instrumentales uniformizados, SOV restaurado -->
```

**Gloss didáctica (palabra por palabra):**

| Morfema | Categoría / Front | Gloss |
|---------|-------------------|-------|
| `o` | partícula (§1.7 vocativo, F4) | vocativo, invocación dirigida |
| `Yaho` | nombre propio (lexicón `onomastica`) | YHWH, el nombre revelado |
| `,` | ortográfico | pausa ceremonial |
| `qadesh` | verbo ritual (F5, lexicón `magia_y_mitologia`) | consagrar, hacer sagrado |
| `altara` | sustantivo instrumental (F5, lexicón `magia_y_mitologia`) | altar (mesa o piedra de ofrenda) |
| `-ka` | postposición de caso (§1.4, dativo) | caso dativo, "al altar" (destino de la consagración) |
| `kár-` | raíz verbal (lexicón `verbos_basicos`) | hacer, realizar |
| `-a-h-wah-nam-` | registro hierático completo (§3.7.1 + §2.1 + §2.2) | hierático (`-h`) + evidencial revelado (`-wah`) + reverencia (`-nam`): cadena ritual obligatoria |
| `-rab` | actitudinal coercitivo (§5.3, F3 "vinculante") | atadura, ligadura ritual (sello permanente) |
| `-nam` | actitudinal (§2.2, "reverencia") | reverencia ante lo sagrado (obligatorio en registro hierático, §3.7.1) |
| `-poi` | performativo (§5.2) | modo declarativo performativo |
| `,` | ortográfico | pausa ceremonial |
| `lampa` | sustantivo instrumental (F5) | lamparilla, vela ritual (llama votiva) |
| `virga` | sustantivo instrumental (F5) | varita ritual (proyección de energía) |
| `-ka` | postposición caso (§1.4, instrumental) | caso instrumental, "mediante la varita" |
| `athama` | sustantivo instrumental (F5) | athame, daga ritual de doble filo |
| `-ka` | postposición caso (§1.4, instrumental) | caso instrumental, "mediante la daga" |
| `tunika` | sustantivo instrumental (F5) | túnica ritual (vestidura ceremonial) |
| `-ka` | postposición caso (§1.4, instrumental) | caso instrumental, "mediante la túnica" |
| `,` | ortográfico | pausa ceremonial |
| `halel` | interjección (lexicón `interjecciones_y_exclamaciones`) | ¡aleluya! alabanza, júbilo |
| `.` | fin de oración | cierre con cadencia sellante (fonología §9) |

**Gloss corrida:** "¡Oh YHWH, al altar, consagrar hieráticamente con visión revelada y reverencia, yo ato vinculante y performativamente! [Mientras] la lamparilla, la varita, la daga y la túnica ritual [son usadas como instrumentos]. ¡Aleluya!"

**Notas gramaticales:**

- **§1.7 vocativo (F4):** la partícula `o` marca la dirección invocativa. `Yaho` es el nombre propio revelado (no se traduce; KFA preserva los nombres sagrados de otras religiones sin sustituir, §1.5). La primera coma separa el vocativo del acto ritual principal; la cláusula invocativa es **autónoma prosódicamente** (curva rise-fall en `o Yaho,` antes de la cláusula principal).
- **§3.18.4 cross-ref (F5):** los instrumentos rituales invocados son `altara` (centro espacial del rito), `lampa` (llama votiva mantenida), `virga` (instrumento de proyección), `athama` (daga de consagración/destierro, no arma), `tunika` (vestidura ceremonial). Todos llevan postposición instrumental `-ka` (§1.4) por uniformidad: `lampa-ka virga-ka athama-ka tunika-ka` = "mediante la lamparilla, la varita, la daga y la túnica".
- **§3.18.7.b invocación operativa (F3 + F5):** la firma ritual operativa sigue el patrón `o [entidad], [objeto-dat], [verbo operativo] [verbo-flex] [actitudinal coercitivo] -poi`. Aquí: `altara-te qadesh kár-a-h-wah-nam-rab-poi` = "al altar, consagrar, hacer-jerárquico-revelado-reverente-vincular-performativo". El dativo `-te` marca el altar como destino de la consagración (§1.4). El evidencial `-wah` (revelado, §2.1) es obligatorio en contexto ritual. El registro hierático exige `-h` + `-nam` (§3.7.1). El actitudinal `-rab` (vinculante) es el más fuerte de los cinco (§5.3 jerarquía: `-ran` < `-ro` < `-ro-hék` < `-rak` < `-rab`); crea **ligadura ritual permanente**, apropiado para el momento de consagración del altar.
- **§5.3 actitudinal `-rab` (F3):** la cadena verbal completa `kár-a-h-wah-nam-rab-poi` cubre los 6 slots modales del verbo ritual hierático: hacer + tiempo presento + jerárquico (`-h`) + revelado (`-wah`) + reverencia (`-nam`) + vincular (`-rab`) + performativo (`-poi`). La combinación `-a-h-wah-nam-rab` es exclusiva de registros hieráticos con evidencial revelado y actitudinal coercitivo vinculante.
- **F5 cierre con `halel`:** la interjección `halel` (¡aleluya!) cierra la invocación con júbilo. En el lexicón está en `interjecciones_y_exclamaciones` con marca de cadencia sellante. La presencia de `halel` al final es **optativa ritual** (cierre celebrativo); el rito es performativamente completo ya con el `-poi` del verbo principal.
- **Cross-ref:** fonología §9 (cadencia sellante, 5 modos de fonación), §10 (silabificación de bījas y derivados), §11 (matriz fonosemántica, por si se generan bījas específicos para altar/lámpara/varita), gramática §1.4 (postposiciones de caso), §1.5 (preservación de nombres sagrados), §1.7 (vocativo), §3.5.5 (apócope derivativa), §3.6.1 (sufijos derivativos), §3.7.1 (registro hierático), §3.13 (evidenciales), §3.18.4 (títulos de Capas III-IV), §3.18.7.b (invocación operativa), §5.2 (performativo), §5.3 (actitudinales coercitivos, jerarquía), lexicón F5 (`qadesh`, `altara`, `lampa`, `virga`, `athama`, `tunika`, `halel`), F3 (`-rab`, `-poi`).

*Estas dos invocaciones (1 y 2) son las **firmas rituales canónicas de v1.3**. La gramática las cita en §1.7, §3.19.7, §3.18.4 y §3.18.7.b; la fonología las работет como ejemplos en §9.3 (curva vocativa), §11.2 (worked example A, bīja `raka`) y §11.0-§11.4 (sistema fonosemántico generativo); el lexicón las respalda con las 20 entradas nuevas de F3 + F4 + F5 (4 actitudinales, 1 partícula, 15 instrumentos/operaciones). Cualquier invocación ritual v1.3 debe poder descomponerse en estos mismos componentes.*
